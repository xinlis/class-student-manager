# 班级座位与考勤系统

这是一个纯前端本地存储的班级座位与考勤系统，适合通过 GitHub Pages 静态发布。

## 数据说明

- 座位表、考勤、备注、轮换设置等保存在浏览器 localStorage。
- 换网址或换浏览器后，需要先在旧网址导出备份 JSON，再在新网址导入备份。
- GitHub 仓库源码不包含学生个人数据。

## 使用

打开 GitHub Pages 发布地址后，点击“导入备份”恢复原站点导出的 JSON 数据。

## 本次版本

- 支持自定义座位分组，例如 `3,4,3,4`。
- 支持手动右移 2 个座位。
- 支持自定义自动轮换间隔天数。
- 自动轮换时间固定为下午 5:30，并显示下次轮换倒计时。
